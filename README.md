# Spotify Tracks Audio Features Analysis Project(Python & Power Bi)

## Overview

This project involves analyzing a Spotify dataset to uncover insights about music trends, artist performance, and audio features. The project demonstrates the full data analytics workflow: data loading, exploration, cleaning, visualization, and dashboard reporting.

## Dataset

**Source:** Kaggle / Spotify dataset

**Description:** The dataset contains information about songs, including audio features (energy, tempo, danceability), popularity, genre, artist, and release year.

**Size:** [50000 rows and 25 columns]

## Tools & Technologies

**Python:** For data loading, cleaning, and Exploratory Data Analysis (EDA)

Pandas / NumPy / Matplotlib / Seaborn: Data manipulation and visualization

**Power BI:** Interactive dashboard creation

**CSV:** Data storage and preprocessing

## Project Steps

**1.Data Loading**

-   Imported the dataset using Python (pandas)
  
-   Checked for missing or null values


**2.Exploratory Data Analysis (EDA)**

-   Analyzed distribution of popularity, duration, and audio features
   
-   Identified patterns and trends across artists, genres, and release years


**3.Data Cleaning**

-   Removed duplicates and irrelevant columns
  
-   Handled missing values


**4.Data Transformation**

-   Created new columns (e.g., popularity categories, decade of release)
   
-   Aggregated data for artist and genre analysis


**5.Dashboard Creation (Power BI)**

**Developed interactive visuals for:**

-   Overall metrics (total tracks, average popularity, top artists/genres)
   
-   Audio feature vs popularity insights
  
-   Music trends over decades
   
-   Added key insights summary for business understanding


**6.Report & Insights**
  
-   Highlighted patterns in track popularity, genre performance, and artist trends
   
-   Created a final report to communicate actionable insights


**Dashboard / Visualizations**

**The Power BI dashboard includes:**

-   **KPIs Overview:** Total tracks, total artists, average popularity
  
-   **Genre & Artist Analysis:** Top genres and top artists by popularity
  
-   **Audio Features vs Popularity:** Energy, danceability, and tempo insights
  
-   **Trends:** Track releases and average popularity across decades




**Key Insights**
-   Tracks with higher energy tend to be more popular
   
-   Pop and Hip-Hop dominate the platform in both track count and popularity
    
-   Number of tracks released has growing steadily over recent decades
    
-   Certain top artists contribute disproportionately to overall popularity
    
-   Audio features like tempo and danceability influence track success
